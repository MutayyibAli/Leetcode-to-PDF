Brute Force Approach: Splitting the String

This approach relies on built-in string manipulation functions to simplify the problem. The core idea is to split the string into individual words and then find the length of the very last word in the resulting collection. While this is often simple to implement, it can be less efficient in terms of memory usage.

  1. Split the String: The input string s is parsed, and all sequences of non-space characters (words) are separated. This is typically done using a built-in split function or a stringstream.
  2. Identify the Last Word: After splitting, the words are stored in a list or are processed sequentially. We identify the last word from this sequence.
  3. Return Length: The length of this last word is calculated and returned.

This approach has a time complexity of O(N), where N is the length of the string, because the entire string must be scanned to be split. The space complexity is also O(N) in the worst case, as a new data structure might be needed to store all the words from the string.

#include <string>
#include <sstream>

class Solution {
public:
    int lengthOfLastWord(std::string s) {
        std::stringstream ss(s);
        std::string word = "";
        std::string lastWord = "";

        // The '>>' operator extracts whitespace-separated words.
        // We loop through all words, and 'lastWord' will hold the last one found.
        while (ss >> word) {
            lastWord = word;
        }

        return lastWord.length();
    }
};

Optimized Approach: Single Pass from the End

This is a more efficient approach that avoids allocating extra memory. By scanning the string from right to left, we can find the last word and calculate its length in a single pass without storing all the words.

  1. Find End of Last Word: Start iterating from the end of the string. Skip any trailing spaces to find the last non-space character, which is the end of the last word.
  2. Count Characters: Once the last character of the word is found, start counting backwards.
  3. Stop Counting: Continue counting until a space is encountered or the beginning of the string is reached. This count represents the length of the last word.
  4. Return Count: The final count is the result.

This approach has a time complexity of O(N) as it involves a single traversal of the string, and a space complexity of O(1) since it only uses a few variables for counting and indexing.

#include <string>

class Solution {
public:
    int lengthOfLastWord(std::string s) {
        int length = 0;
        int i = s.length() - 1;

        // 1. Skip all trailing spaces
        while (i >= 0 && s[i] == ' ') {
            i--;
        }

        // 2. Count the length of the last word
        while (i >= 0 && s[i] != ' ') {
            length++;
            i--;
        }

        return length;
    }
};

Pythonic solution for best approach

The best approach is the single pass from the end, which is optimal in terms of space. The following Python code implements this logic in a clean and readable way.

class Solution:
    def lengthOfLastWord(self, s: str) -> int:
        length = 0
        # We start iterating from the last character of the string
        for i in reversed(range(len(s))):
            # If the character is not a space, it's part of a word
            if s[i] != ' ':
                length += 1
            # If we encounter a space after we've started counting,
            # it means we have found the end of the last word.
            elif length > 0:
                return length

        # This handles cases where the string has no leading spaces,
        # e.g., "word" or "another word".
        return length
