Brute Force Approach (Using Stringstream)

This method utilizes extra space to store the words and build the result. A stringstream is used to easily parse the input string, which conveniently handles splitting by spaces, including multiple spaces between words.

  1. Initialize a stringstream with the input string s.
  2. Create a string variable, word, to hold each extracted word, and a result string to build the final output.
  3. Read words from the stringstream one by one. The stream extraction operator >> automatically splits the string by whitespace.
  4. For each word read, prepend it to the result string. To ensure single-space separation, a space is also prepended before adding the word (except for the very first word).
  5. This process naturally reverses the order of words. The final result string is the desired output.

This approach is straightforward and easy to read but requires additional memory to store the words and the new string. The time complexity is O(N) because we iterate through the string to parse it and then build a new string, and the space complexity is O(N) for storing the intermediate words and the final result.

#include <sstream>
#include <string>
#include <vector>
#include <algorithm>

class Solution {
public:
    std::string reverseWords(std::string s) {
        std::stringstream ss(s);
        std::string word;
        std::string result = "";

        // Read words from the stringstream and prepend to the result
        while (ss >> word) {
            if (result.empty()) {
                result = word;
            } else {
                result = word + " " + result;
            }
        }

        return result;
    }
};

In-place Reversal Approach (Two Pointers)

This optimal approach modifies the string in-place, achieving the result with O(1) extra space. It's a clever multi-step process involving reversing different parts of the string.

  1. Reverse the Entire String: The first step is to reverse the whole string. This puts the words in their correct final order, but the letters within each word are now backward (e.g., "the sky is blue" becomes "eulb si yks eht").
  2. Clean Spaces and Reverse Each Word: Iterate through the string with two pointers: a read pointer (i) to scan the current string and a write pointer (right) to build the clean, final string at the beginning of the same character array.
    • As you scan with i, skip any spaces.
    • When a word is found, copy its characters from i to the position starting at right.
    • After a word is copied, its characters (e.g., "eulb") are now in the correct position. Reverse this specific word segment to restore its original spelling (e.g., "blue").
    • Append a single space after the corrected word using the right pointer.
  3. Final Cleanup: After iterating through all the words, the string might have a trailing space. Resize the string to its correct final length to remove this space and any other junk characters at the end.

This method is highly efficient as each character is processed a constant number of times. The time complexity is O(N), and since all modifications happen on the input string itself, the space complexity is O(1).

#include <string>
#include <algorithm>
#include <vector>

class Solution {
public:
    std::string reverseWords(std::string s) {
        // Step 1: Reverse the entire string
        reverse(s.begin(), s.end());

        int n = s.size();
        int left = 0;
        int right = 0;
        int i = 0;

        while (i < n) {
            // Skip leading/multiple spaces from the source
            while (i < n && s[i] == ' ') {
                i++;
            }
            if (i == n) {
                break;
            }

            // Copy the word from read position (i) to write position (right)
            while (i < n && s[i] != ' ') {
                s[right++] = s[i++];
            }

            // Step 2: Reverse the current word which is now at [left, right-1]
            reverse(s.begin() + left, s.begin() + right);

            // Add a single space after the word
            s[right++] = ' ';
            left = right;
        }

        // Step 3: Resize the string to remove the trailing space
        if (right > 0) {
            s.resize(right - 1);
        } else {
            s.resize(0); // Handle case of empty or all-space string
        }

        return s;
    }
};

Pythonic Solution

The most idiomatic and concise Python solution leverages built-in string and list methods. It chains together strip(), split(), list reversal, and join() to achieve the result in a single, readable line. While this approach uses O(N) space for the list of words, it is the preferred and most practical solution in Python.

  1. s.strip(): Removes any leading or trailing whitespace from the string.
  2. .split(): Splits the string by any whitespace and returns a list of words. It automatically handles multiple spaces between words.
  3. [::-1]: This slice notation reverses the list of words.
  4. " ".join(...): Joins the words from the reversed list into a new string, with a single space as the separator between them.
class Solution:
    def reverseWords(self, s: str) -> str:
        # 1. Strip leading/trailing spaces
        # 2. Split the string by spaces into a list of words
        # 3. Reverse the list of words
        # 4. Join the words back with a single space
        return " ".join(s.strip().split()[::-1])
